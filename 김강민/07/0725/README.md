## [브론즈 5] X보다 작은 수

[문제 보러가기] https://www.acmicpc.net/problem/10871

## 문제 설명

정수 N개로 이루어진 수열 A와 정수 X가 주어진다. 이때, A에서 X보다 작은 수를 모두 출력하는 프로그램을 작성해라.

### 꼭 알아야하는 것

-   리스트
-   map -> 아직 map을 잘 모르게따... 공부하자..

## 해설

1. N과 X를 입력받는다
2. 리스트 A 를 입력받는다.
3. X보다 작을경우 그 숫자를 출력하고, end = " " 를 통해 공백을 삽입한다.
